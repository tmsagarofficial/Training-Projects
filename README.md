## 🏦 S-Bank (JAR + JDBC App)

A simple banking application built using Java and JDBC. It supports basic operations like account creation, deposit, withdrawal, and transaction history.

- **Tech Stack**: Java, JDBC, MySQL
- **Features**:
  - User authentication
  - Account management
  - Transaction logging
